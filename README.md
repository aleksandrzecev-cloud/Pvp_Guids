## ğŸ¨ Customization

The website uses inline CSS and JavaScript (no external dependencies except Three.js for the 3D viewer in the blueprint tool). To customize:

### Colors
Edit the CSS `:root` or color values in the `<style>` section:
- Primary accent: `#e94560` (red/pink)
- Secondary: `#00d4ff` (cyan)
- Background: `linear-gradient(135deg, #1a1a2e 0%, #16213e 100%)`

